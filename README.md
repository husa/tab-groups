# tab-groups

[![Build Status](https://travis-ci.org/husa/tab-groups.svg?branch=master)](https://travis-ci.org/husa/tab-groups)

## Local development

```
yarn start
```

## Build
Assets build
```
yarn build
```
Build icons
```
cd src/assets/icons && node generate.js
```

Complete build
```
yarn lint
yarn flow
yarn test
yarn build
```

Release
```
yarn release 1.2.3
```
